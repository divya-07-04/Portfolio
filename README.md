# 🌐 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built to showcase my skills, projects, education, certifications, and professional profile as a Software Developer.

## 🚀 Live Demo

🔗 **Portfolio:** https://divya-07-04.github.io/Portfolio/

## 📌 Features

* ✨ Modern and responsive UI
* 🌙 Dark & Light mode
* 🎯 Animated typing effect
* 🎇 Interactive particle background
* 📜 Smooth scrolling navigation
* 📱 Mobile-friendly design
* 📂 Downloadable resume
* 🔗 GitHub repository integration using GitHub API
* 🎨 Scroll animations powered by AOS
* ⬆️ Scroll-to-top button
* 📬 Contact form
* ⚡ Fast loading experience with loading animation

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### Libraries & APIs

* Font Awesome
* Google Fonts (Poppins)
* AOS (Animate On Scroll)
* tsParticles
* GitHub REST API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/divya-07-04/Portfolio.git
```

Navigate to the project folder:

```bash
cd Portfolio
```

Open the project:

* Open `index.html` directly in your browser, or
* Run the project using the VS Code Live Server extension.
